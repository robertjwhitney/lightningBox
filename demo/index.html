Lightning Box

Lightning Box is a CSS/jQuery light box with the goal of staying super light weight. It does this by:

  • Relying on CSS techniques developed by Jenna Smith and Eric Meyer
  • Maintaining a limited feature set
  • Trying to keep the js efficient
